Creating virtual environment:
    - python -m venv .venv
    : Uppdate interpretor in virtual environment(ctrl+p) and select the interpreter in virtual env
        - Python: Select Interpreter
    : Open new Terminal from top toolbar, the execution point may look like '(.venv) PS D:\Git2\Eclipse\Cashflow>'
        


Create final package list
    - pip freeze > requirements.txt
Other computer install packages
    - pip install -r requirements.txt
